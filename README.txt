== Installation ==

To install, you just need to follow these steps

1. Upload the zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the Settings page and configure.

== Frequently Asked Questions ==

= Will the guest post publish automatically? =

You can choose to let it publish automatically or save as draft. If you save it as draft, you will need to vet the content in the database automatically.
